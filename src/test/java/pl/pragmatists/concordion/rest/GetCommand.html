Get Command

Get command sends request to server

Sends request to server

When following fixutre is executed: 
<rest:get>/hello</rest:get>
Then GET request should be set to /hello

Verifies response status (success)

Given server responds with 200 for GET on /hello When following fixutre is executed: 
<rest:get>/hello</rest:get>
<rest:status>HTTP/1.1 200 OK</rest:status>
Then result is success

Verifies response status (failure)

Given server responds with 404 for GET on /hello When following fixutre is executed: 
<rest:get>/hello</rest:get>
<rest:status>HTTP/1.1 200 OK</rest:status>
Then result is failure
